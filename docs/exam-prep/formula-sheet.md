# 📐 Macro Formula Sheet

*All key formulas for ECON1002 / ECOS2002 — print-friendly revision reference*

---

## National Accounts

| Formula | Description |
|---|---|
| **GDP = C + I + G + NX** | Expenditure approach |
| **NX = X − M** | Net exports |
| **Real GDP = Nominal GDP / Price level × 100** | Removing inflation |
| **GDP growth = (GDP₁ − GDP₀) / GDP₀ × 100** | Annual growth rate |
| **GNI = GDP + Net factor income from abroad** | Gross National Income |
| **NNI = GNI − Depreciation** | Net National Income |

---

## Inflation & Price Level

| Formula | Description |
|---|---|
| **CPI = (Cost of basket / Base year cost) × 100** | Price level index |
| **Inflation rate = (CPI₁ − CPI₀) / CPI₀ × 100** | % change in CPI |
| **Real interest rate = Nominal rate − Inflation** | Fisher equation (approx.) |
| **%ΔM + %ΔV = %ΔP + %ΔY** | Quantity theory (growth form) |
| If V, Y constant: **%ΔM = %ΔP** | Money → Inflation (long run) |

---

## Unemployment

| Formula | Description |
|---|---|
| **Unemployment rate = Unemployed / Labour force × 100** | Standard definition |
| **Labour force = Employed + Unemployed** | (Not including "not in labour force") |
| **Participation rate = Labour force / Working-age population × 100** | |
| **Okun's Law: %ΔGDP ≈ −2 × Δunemployment rate** | Output-unemployment link |
| **Output gap = (Y − Y\*) / Y\* × 100** | % deviation from potential |

---

## Keynesian Cross / Multiplier

| Formula | Description |
|---|---|
| **C = C̄ + c(Y − T)** | Consumption function |
| **PAE = C + Iₚ + G + NX** | Planned aggregate expenditure |
| **Y\* = A / (1 − c + m)** | Equilibrium income (open economy) |
| **Multiplier k = 1 / (1 − c + m)** | Spending multiplier |
| **k_G = 1 / (1 − c + m)** | Government spending multiplier |
| **k_T = −c / (1 − c + m)** | Tax multiplier (negative) |
| **Balanced budget multiplier = 1** | ΔG = ΔT → ΔY = ΔG |

*Typical USYD values: c = 0.8, m = 0.1 → k = 1 / 0.3 = 3.33*

---

## Money & Banking

| Formula | Description |
|---|---|
| **Money multiplier = 1 / rr** | rr = reserve ratio |
| **ΔM = (1/rr) × ΔReserves** | Deposit expansion |
| **MV = PY** | Equation of exchange |

---

## IS-LM Model

| Formula | Description |
|---|---|
| **IS curve: i = a − b×Y** | Downward sloping (goods market) |
| **LM curve: i = c×Y − d** | Upward sloping (money market) |
| **Policy reaction (Taylor Rule): i = r\* + π + 0.5(π − π_T) + 0.5 × output gap** | RBA reaction function |

---

## Phillips Curve

| Formula | Description |
|---|---|
| **π = π_e − α(u − u_n)** | Expectations-augmented PC |
| **NAIRU (u_n) ≈ 4.0–4.25%** (Australia post-COVID) | Non-accelerating inflation rate |
| **Sacrifice ratio = % output lost per 1pp fall in inflation** | Cost of disinflation |

---

## Solow Growth Model

| Formula | Description |
|---|---|
| **y = k^α** | Per-worker production function |
| **Steady state: s×f(k) = (δ + n + g)k** | Saving = break-even investment |
| **k\* = (s / (δ+n+g))^(1/(1−α))** | Steady-state capital per worker |
| **Golden rule: s_GR = α** | Capital-share saving rate |
| **Australia: s≈0.25, α≈0.35, δ+n+g≈0.08 → k\*≈5.77** | Calibration |

---

## Open Economy / Balance of Payments

| Formula | Description |
|---|---|
| **CA + KFA = 0** (approximately) | BOP identity |
| **(S − I) + (T − G) + (M − X) = 0** | Sectoral balances identity |
| **Real exchange rate = e × (P\* / P)** | Competitiveness measure |
| **Purchasing Power Parity: e = P / P\*** | Long-run exchange rate |
